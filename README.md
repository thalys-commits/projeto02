 Projeto 02

Este repositório contém o **Projeto 01**, desenvolvido como parte de um trabalho da faculdade.  
O objetivo é praticar conceitos de **HTML, CSS e JavaScript**, criando um site interativo com formulário de cadastro e validação de senha.

---

## 🚀 Funcionalidades

- Estrutura de páginas com **HTML semântico**
- Estilização responsiva com **CSS**
- Formulário de contato com:
  - Validação de campos obrigatórios
  - Regras de senha (mínimo 8 caracteres, letra maiúscula, número e caractere especial)
  - Barra de força da senha
  - Checklist de requisitos dinâmico
  - Confirmação de senha com feedback visual
